Alittle background about us & this project..

This project contains copyright infringement (hence the Blue Moon™ & Five Guys™ logo) and is not to be taken seriously, no profit has or will be made off of this web application.

How did we get the name of our project and the name of our team in the first place? The name came about from our Project Manager, Brandell, who at the time was coming up with the name while eating dinner. His dinner consisted of Five Guys'™ burgers accompanied with Blue Moon™ beer, you can see where this goes. The purpose of Blue Moon Suites was to fulfill a project for our Software Engineering CSC4350 class which was to create an application with many requirements that this project fulfills. Blue Moon Suites is a hotel management system that was designed to be a web application for the goal of being external (customer-facing) as well as internal like the majority of hotel management systems.
